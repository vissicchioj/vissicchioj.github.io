\section{Questions}
\subsection{What is the relationship between a guest operating system and a host operating system like VMware? What factors need to be considered in choosing the host operating system?}

The host operating system logically creates the illusion that the processor has its own processor and virtual memory. It connects with the hardware. The guest operating system is provided with its own virtual copy of the underlying computer that connects to the host operating system. What needs to be considered in choosing the host operating system is its system calls in order to produce system programs which are most important from a user's point of view.
